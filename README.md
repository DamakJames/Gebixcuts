```
[ 01. DISCOVER ] ───────> [ 02. PLAN ] ───────> [ 03. CREATE ] ───────> [ 04. DELIVER ]
Understand vision,        Develop strategy,       Shoot & produce with    Color grade, edit,
audience & objectives     schedule & workflow     cinema-grade gear       review & final delivery
```

1. **Discover:** We begin by deeply understanding your goals, audience, and creative vision to ensure every frame aligns with your objectives.
2. **Plan:** Our production team develops a thorough creative strategy, shot list, production schedule, and execution roadmap.
3. **Create:** Using professional cinema camera systems, lighting packages, and drone technology, we capture and direct your story with precision.
4. **Deliver:** Your project undergoes meticulous post-production, professional color grading, audio mastering, and on-time high-definition delivery.

## 🎨 Design System & Visual Architecture

The Gebixcuts digital presence is built upon a **Cinematic Obsidian & Gold (`#0A0A0A` + `#E6AF2E`)** dark-mode design system. Inspired by high-end film production consoles and luxury cinema aesthetics, the platform emphasizes rich contrast, warm metallic accents, glassmorphic surfaces, and responsive fluid motion.

```css
:root {
  /* Core Palette */
  --bg-obsidian: #08080A;
  --bg-surface: #121216;
  --bg-card-hover: #1A1A22;
  
  /* Brand Accents */
  --gold-primary: #E6AF2E;
  --gold-glow: rgba(230, 175, 46, 0.15);
  
  /* Typography */
  --text-primary: #F8F9FA;
  --text-secondary: #A0A6B2;
  
  /* Borders & Glassmorphism */
  --border-glass: rgba(255, 255, 255, 0.08);
}
```

## 💻 Getting Started (Local Development)

### Prerequisites
- [Node.js](https://nodejs.org/) (v18.0 or higher)
- [npm](https://www.npmjs.com/) or [pnpm](https://pnpm.io/)

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/DamakJames/Gebixcuts.git
   cd Gebixcuts
   ```

2. **Install project dependencies:**
   ```bash
   npm install
   # or
   pnpm install
   ```

3. **Start the local development server:**
   ```bash
   npm run dev
   # or
   pnpm dev
   ```

4. **Preview locally:**
   Open [http://localhost:3000](http://localhost:3000) (or `http://localhost:5173`) in your web browser to view and interact with the Gebixcuts cinematic experience.
